# mysql-student-records-queries
A collection of basic MySQL queries demonstrating database creation, data retrieval, filtering, sorting, aggregation, grouping and SQL functions using a students records database.

 #Overview

This project contains basic MySQL queries used to explore and analyze a student records database. It demonstrates fundamental SQL concepts including data retrieval, filtering, sorting, aggregate functions, grouping, and conditional queries.

#Topics Covered

- Creating and selecting a database
- Retrieving records with SELECT
- Filtering data using WHERE
- Selecting unique values with DISTINCT
- Pattern matching using LIKE
- Using comparison operators
- Logical operators (AND, OR)
- Sorting results with ORDER BY
- Limiting output using LIMIT
- Aggregate functions
  - AVG()
  - MAX()
  - MIN()
- Grouping data using GROUP BY
- Filtering grouped results using HAVING

## Technologies Used

- MySQL
- MySQL Workbench

## Sample Queries

Examples included in this project:

- Display all student records
- Retrieve female and male students
- Find students with final exam scores above 80
- Display students by year of study
- Calculate average final exam scores
- Find highest assignment score
- Find lowest attendance percentage
- Calculate average scores by course
- Display courses with average final exam scores above 70

## Learning Outcome

This project was completed as part of learning SQL database querying and data analysis using MySQL Workbench.
